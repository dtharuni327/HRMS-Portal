# 🚀 HRMS Portal

A Human Resource Management System (HRMS) built with modern web technologies.

---

## ⚙️ Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- React Router
- Axios (API calls)
- Zustand / React Query (state management)
- JWT Authentication
